Notes by Saeran Vasanthakumar --- 22/07/01 - Cause and Effect Chaining in Tamil Grammar Something that has bothered me for a while, is the way complex Tamil sentences structure causality in a very idiosyncratic way, to me. By causality here, I mean reasoning about the relationships between a cause and it’s subsequent effect. The subsequent effect can be another cause, which then propogates a new effect, and so causal reasoning about complex topics can occur when chaining together multiple preceding causes and effects as a sort of heavily composed function (effect_n(…(effect_2(effect_1(cause_1)))…). The specific grammar differences I am refering to is the way Tamil inverts the subject, verb, object order of a clause. Here’s I’ll parse a single clause “the heat caused a crisis” with a clear cause-effect structure. $$ \begin{alignat}{7} &\textbf{English Single Clause} \\ \\ &( \text{ subject } && &\cdot &\text{ verb } && &\cdot &\text{ object } ) \\ \\ &( \text{ The heat } && &\cdot &\text{ caused } && &\cdot &\text{ a crisis }) \\ \\ &( X_1 && &\cdot &\mapsto && &\cdot &X_2) \\ \\ &\textbf{Tamil Single Clause} \\ \\ &( \text{ subject } && &\cdot &\text{ object } && &\cdot &\text{ verb }) \\ \\ &( \text{ வெப்பம் } && &\cdot &\text{ நெருக்கடியை } && &\cdot &\text{ ஏற்படுத்தியது }) \\ \\ &( X_1 && &\cdot &X_2 && &\cdot &\mapsto ) \\ &\color{white}((\text{ வெப்பம் } && &\color{white}\cdot &\color{white}\text{ நெருக்கடியை } && &\color{white}\cdot &\color{white}\text{ ஏற்படுத்தியது }) && &\color{white}\cdot &\color{white}\text{ நெருக்கடிகளை } && &\color{white}\cdot &\color{white}\text{ உருவாக்குகிறது }) \\ \end{alignat}$$ Does the subject-verb-object inversion leads to inverting the causality of the sentence as multiple clauses are involved? Here's a simple, maybe trivial example. I want to add a subordinate clause that uses the effect of the previous sentencs as a new cause - “the heat caused a crisis, eventually creating more crisises.” $$ \begin{alignat}{6} &\textbf{English Multi-clause} \\ \\ &(( \text{ subject } && &\cdot &\text{ verb } && &\cdot &\text{ object } ) && &\cdot &\text{ verb } && &\cdot &\text{ object }) \\ \\ &(( \text{ The heat } && &\cdot &\text{ caused } && &\cdot &\text{ a crisis }) && &\cdot &\text{ creating } && &\cdot &\text{ more crises }) \\ \\ &(( X_1 && &\cdot &\mapsto && &\cdot &X_2) && &\cdot &\mapsto && &\cdot &X_3) \\ \\ &\textbf{Tamil Multi-clause} \\ \\ &((\text{ subject } && &\cdot &\text{ object } && &\cdot &\text{ verb }) && &\cdot &\text{ object } && &\cdot &\text{ verb }) \\ \\ &((\text{ வெப்பம் } && &\cdot &\text{ நெருக்கடியை } && &\cdot &\text{ ஏற்படுத்தியது }) && &\cdot &\text{ நெருக்கடிகளை } && &\cdot &\text{ உருவாக்குகிறது }) \\ \\ &((X_1 && &\cdot &X_2 && &\cdot &\mapsto ) && &\cdot &X_3 && &\cdot &\mapsto) \\ \end{alignat} $$ Here we can see how cause and effect is chained in both languages, with the object from the first clause, acting as subject in the second clause, and thus switching from acting as the effect to the cause, respectively. In English, every cause is a bridge between related effects. In Tamil, the cause seemingly bridges between the "wrong" effects. --- email: saeranv @ gmail dot com git: github.com/saeranv
